# Server Monitoring Dashboard

A real-time server monitoring dashboard built using Python, Django, REST APIs, and psutil.

## Features

- Live CPU Monitoring
- Memory Usage Tracking
- Disk Usage Monitoring
- Network Monitoring
- Battery Status
- Running Processes Count
- Real-Time Auto Refresh Dashboard
- Interactive Charts using Chart.js

## Technologies Used

- Python
- Django
- REST API
- psutil
- HTML/CSS
- JavaScript
- Chart.js

## Installation

```bash
git clone https://github.com/yourusername/Server-Monitoring-Dashboard.git
```

```bash
cd Server-Monitoring-Dashboard
```

```bash
pip install -r requirements.txt
```

```bash
python manage.py runserver
```

## Output

The dashboard displays real-time system performance statistics.
